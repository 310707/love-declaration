TROUBLESHOOT MUSIK TIDAK PLAY:
==============================

Jika musik tidak bisa diputar di website:

1. Format Check:
   - Ensure MP3 format (most compatible)
   - Try convert WAV/OGG ke MP3
   
2. Size Check:
   - File terlalu besar (> 50MB)?
   - Compress dengan ffmpeg atau online tools
   
3. Server CORS:
   - Jika dari external server
   - Cek CORS headers
   - Try ganti jadi MP3 di-host di folder assets

4. Browser Support:
   - Try berbagai browser
   - Check browser console error
