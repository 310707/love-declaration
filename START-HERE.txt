🆘 QUICK HELP:

❌ Website tidak terbuka?
   → Pastikan membuka index.html (bukan file lain)
   → Try buka dengan browser berbeda
   → Cek file ada di folder yang benar

❌ Foto tidak muncul?
   → Coba upload ulang
   → Pastikan format JPG/PNG
   → Cek ukuran file < 5MB

❌ Musik tidak dengar?
   → Refresh browser (Ctrl+F5)
   → Cek volume browser/device
   → Cek file musik ada di folder assets

❌ CSS tidak applied?
   → Hard refresh: Ctrl+Shift+Delete, clear cache
   → Cek nama file style.css sama di HTML
   → Pastikan file tidak di-rename

❌ Animasi berjalan lambat?
   → Normal di device lama
   → Tutup browser lain
   → Baca TROUBLESHOOT.txt
